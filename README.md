# Password Strength Checker

A simple and responsive Password Strength Checker built using HTML, CSS, and JavaScript.

The application analyzes a user's password in real time and provides instant feedback based on common password security requirements.

---

## Features

- Real-time password strength analysis
- Checks password length
- Detects uppercase letters
- Detects lowercase letters
- Detects numbers
- Detects special characters
- Visual strength indicator
- Show/Hide password button
- Copy password button
- Responsive user interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## Password Rules

The application checks whether the password contains:

- At least 8 characters
- One uppercase letter
- One lowercase letter
- One number
- One special character

Based on these checks, the password is rated from:

- Very Weak
- Weak
- Fair
- Good
- Strong

---

## How to Run

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Install the Live Server extension (if not already installed).
4. Open `index.html`.
5. Click **Go Live**.
6. Start testing different passwords.
